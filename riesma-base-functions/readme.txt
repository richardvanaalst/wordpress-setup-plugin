== Description ==

Add Riesma's functions for creating custom post types, order and removing admin menu items, etc.


== Installation ==

1. Upload the `riesma-base-functions` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the functions in the plugin's php file to create more post types and edit the preferences
